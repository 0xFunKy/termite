A simple VTE-based terminal.

KEYBINDINGS
===========

Scrollback search:

* ``ctrl-/``: start search (forward)
* ``ctrl-?``: start search (backward)
* ``ctrl-shift-n``: search forward
* ``ctrl-shift-p``: search backward

While typing search pattern:

* ``enter``: start search
* ``escape``: cancel search
